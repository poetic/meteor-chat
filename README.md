# Setup:
1. clone the repo:
```
git clone git@github.com:poetic/meteor-chat.git
```
2. run:
```
curl https://install.meteor.com/ | sh
```
3. start meteor
```
cd meteor-char
meteor
```
4. checkout the app and start hacking!
```
localhost:3000
```
